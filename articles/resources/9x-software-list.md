# 可在Windows 98/ME上运行的最新软件版本

![9x-software-list.jpg](./9x-software-list.jpg)

Windows 98以及我个人最喜爱的ME，永远在我心中占有一席之地（尽管它们在2003年之后的电脑上根本无法正常运行，唉）。和它们的前辈Windows 95不同，这两款系统即便存在诸多限制，也依然具备不少现代化的能力。
本列表中收录的软件仅适用于纯净版Windows 98/ME系统。使用KernelEx的用户可以获得额外优势，能够运行更多2010年代的软件。

**请获取[适用于Windows 9x的非官方Visual C++ 2008运行库]()以获得最佳效果。部分程序需要GDIPLUS.dll文件。**
*
**另外补充一点：软件的便携版通常可以在老旧系统上正常运行。**
